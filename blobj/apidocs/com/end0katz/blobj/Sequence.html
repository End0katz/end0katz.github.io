Interface Sequence<T>

Type Parameters:
T - the type of the items in this sequence.
All Superinterfaces:
Iterable<T>
All Known Implementing Classes:
Sequence.Sequences, Sequence.Sequences.Fibonacci
public interface Sequence<T> extends Iterable<T>
Interface to represent a sequence of any Objects, normally Integers, Doubles or Longs. They can be any length (including infinite).
