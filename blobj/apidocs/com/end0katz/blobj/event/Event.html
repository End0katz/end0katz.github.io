Class Event

java.lang.Object
com.end0katz.blobj.event.Event
public class Event extends Object
Event that you can subscribe to. To add your method to an event, run: eventInstance.add(YourClass::YourMethod). There are more complex methods of doing this (see src/main/resources/docs/Event.md) Creating an event is done through builder()
