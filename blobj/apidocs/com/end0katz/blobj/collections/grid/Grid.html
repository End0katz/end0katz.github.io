Interface Grid<T>

Type Parameters:
T - the type of object in this grid.
All Superinterfaces:
Collection<T>, Iterable<T>, SequencedCollection<T>
All Known Implementing Classes:
ArrayMatrix, Matrix
public interface Grid<T> extends SequencedCollection<T>
The Grid interface is a SequencedCollection whose encounter order is starting at the top left, traversing to the right and going down a row to the left column when reaching end of row.
