Contributing to Blobj

How to Contribute

We welcome contributions from everyone! Follow the steps below to get started:

If you can code:

  1. Fork the repository on GitHub.
  2. Clone your fork to your local machine.
  3. Create a new branch for your changes.
  4. Make your changes and commit them with clear messages.
  5. Push your changes to your fork and open a pull request.

If you cannot code:

Create an issue request with your idea and the label "Enhancement" on the GitHub repository. We will review it and potentially add it. If we have decided not to, it will be marked as "won't fix"
